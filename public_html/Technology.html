Hey this is my Technology page and this is a summary of HTML and CSS.

Html stands for Hypertext Markup Language. Html is used to create webpages you create these web pages by using html tags. You put elements in these tags and they all do different things, for instance using the element "p" makes a paragraph.

Css stands for Cascading Style Sheet. Css is used to styalize your Html webpage by adding different aspects to make it unique. You can do almost everything in css from adding colors to making fonts look different.
